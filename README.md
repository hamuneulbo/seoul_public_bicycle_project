## 프로젝트 설명

데이터베이스를 구축하고, 데이터베이스에 기초한 질의에 맞는 쿼리문을 작성한 프로젝트입니다.  
데이터는 서울시 따릉이 데이터를 사용하기로 결정했으며, 사용자와 회사 입장에서 질의를 생성했습니다.
